# TreTrip FanZone

TreTrip FanZone is a bilingual football platform built with Next.js 16 and Supabase.

## Setup

1. Copy `.env.example` into `.env` and fill in your Supabase values.
2. Install packages with `npm install`.
3. Apply the database schema with `npm run db:apply`.
4. Seed categories and the admin account with `npm run db:seed`.
5. Start the app with `npm run dev`.

## Stack

- Next.js 16 App Router
- React 19
- Supabase Auth
- Supabase Postgres
- TypeScript
